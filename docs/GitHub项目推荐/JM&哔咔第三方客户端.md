## 1、介绍
回归正题，最近我在GitHub挖到这个**Breeze** —— 直接破了官方所有槽点，flutter写的第三方客户端，把JM和哔咔整到一个App里，简直爽爆了

Breeze 是哔咔(PicACG/bika)与禁漫（禁漫天堂，jm）的第三方安卓客户端，采用 Flutter 开发

## 2、这App牛在哪？
**零广告，真干净**  
开屏直接进，阅读中途零弹窗，再也不用看"同城约X"了

**速度起飞**  
图片预加载做得比官方好太多，流畅的一批（其实也就是取决于你的梯子和机场性能）

**下载自由**  
支持批量下载漫画，还能导出成ZIP存本地，再也不用做数学题了

**双平台合一**  
一个App同时登录JM和哔咔，不用来回切了，直接享受双倍快乐

**自动签到**  
自动签到功能，每天打开自动领积分

**Modern UI**  
有暗黑模式，晚上看不伤眼

**闪退**

解决JM老是动不动就闪退的老毛病，并且加载不成功时可以多试几遍，一般不超过5遍就可以看了

## 3、缺点
本人亲自试毒

**收藏点赞要得多点几次**  
不知道是不是接口问题，有时候点收藏或点赞，明明显示成功了，退出去再进发现没点上。我现在养成了点两次的习惯 —— 点一下，等两秒，取消再点一下，这样比较稳。希望作者后续能优化下这个接口同步

**对梯子质量有要求**  
虽然比官方流畅，但如果你梯子节点太拉胯，还是会报错或者图片加载到一半卡住失败。我试过用某些免费节点，时不时跳出"网络错误"，换成付费稳一点的线就没事，最好还是用机场

> 不过也没太多不好用的地方解决屏蔽广告已经是我最大的愿望了，如果遇到了可以说一下
