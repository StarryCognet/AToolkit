# Ollama 安装到 D 盘（方案一）：使用命令行指定路径
## 一、准备工作
1. **下载安装程序**：访问 Ollama 官方网站（[https://ollama.com/download](https://ollama.com/download)），下载适用于 Windows 的安装程序 `OllamaSetup.exe`。

## 二、安装步骤
1. **打开命令提示符**：按 `Win + R` 键打开“运行”对话框，输入 `cmd`，然后按 `Enter` 键，打开命令提示符。

**指定安装路径**：在命令提示符中，运行以下命令来安装 Ollama 并指定安装路径为 D 盘的某个文件夹，例如 `D:\Ollama`：

2. `OllamaSetup.exe /DIR="D:\Ollama"`

**提示**：如果 `OllamaSetup.exe` 文件位于当前目录，可以使用相对路径运行安装程序，例如：

3. `.\OllamaSetup.exe /DIR="D:\Ollama"`

这样可以更灵活地指定安装路径，尤其是在安装程序位于当前目录时。

## 三、验证安装
**验证安装是否成功**：安装完成后，打开命令提示符或 PowerShell，输入以下命令验证安装是否成功：

1. `ollama --version`

如果显示 Ollama 的版本号，则说明安装成功。

## 四、通用注意事项
+ **确保路径格式正确**：在指定安装路径时，确保路径格式正确，避免使用中文字符或特殊符号。
+ **管理员权限**：安装过程中可能需要管理员权限，因此建议以管理员身份运行命令提示符。
+ **修改路径时谨慎操作**：在修改文件路径或命令时要谨慎，避免误删或误改重要文件。
+ **可适用于其他软件安装**：此方法适用于许多 Windows 软件，可以通过命令行指定安装路径，安装到指定的磁盘或文件夹中。
