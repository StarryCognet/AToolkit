# 清浊

## 功能简介

清浊是一款简洁的电脑空间清理工具。

## 主要特点

- 界面简洁美观
- 快速扫描系统垃圾文件
- 清理临时文件和缓存
- 释放磁盘空间
- 硬核清理模式

## 使用场景

- 定期清理系统垃圾文件
- 释放磁盘存储空间
- 提升系统运行速度
- 清理浏览器缓存和历史记录
