# DiskGenius

## 功能简介

DiskGenius 是一款功能强大的磁盘管理及数据恢复软件，被誉为"古希腊掌管电脑数据的神"。

## 主要特点

- 磁盘分区管理（建立、删除、格式化分区等）
- 数据恢复功能（丢失分区恢复、误删除文件恢复等）
- 分区备份与还原
- 磁盘克隆和分区复制
- 分区表错误检查与修复
- 坏道检测与修复
- 支持 GPT 磁盘和 GUID 分区表
- 支持多种文件系统（FAT12/FAT16/FAT32/NTFS/EXT3等）
- 支持各种存储设备（IDE、SCSI、SATA、U盘、存储卡等）

## 使用场景

- 硬盘数据恢复
- 磁盘分区管理
- 系统迁移和克隆
- 数据备份和还原
- 硬盘故障诊断和修复
