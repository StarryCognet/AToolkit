# Anvi Folder Locker

## 功能简介

Anvi Folder Locker 是一个免费的安全工具，旨在帮助用户管理和保护重要文件。

## 主要特点

- 文件和文件夹加密
- 隐藏敏感文件和文件夹
- 设置访问密码保护
- 防止文件被修改、破坏或删除
- 简单易用的操作界面

## 使用场景

- 保护个人隐私文件
- 防止他人访问重要数据
- 隐藏不想被别人看到的文件
- 为机密文件添加额外安全层
