# PotPlayer

## 功能简介

PotPlayer 是一款免费的多媒体播放器，支持各种视频编解码器和格式，是一款功能强大的万能查看工具。

## 主要特点

- 支持多种视频格式（AVI、MPEG、ASF、WMV、M2V、MTS、DPG 等）
- 支持多种音频格式（MP3、WAV、FLAC 等）
- 内置 OpenCodec 支持，易于添加所需编解码器
- 支持硬件加速解码
- 支持多种字幕格式
- 可自定义皮肤和界面

## 使用场景

- 观看各种格式的视频文件
- 音频播放和管理
- 在线流媒体播放
- 视频调试和分析
