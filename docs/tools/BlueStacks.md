# BlueStacks (安卓虚拟机)

## 功能简介

BlueStacks 是一款在电脑上使用的安卓虚拟机，功能全面且使用内存较少。

## 主要特点

- 在 Windows 系统上运行 Android 应用
- 支持触控和键盘鼠标操作
- 可自定义虚拟设备配置
- 支持多开应用
- 与 Google Play 商店集成
- 性能优化，资源占用较低

## 使用场景

- 在电脑上玩 Android 手机游戏
- 使用 Android 版办公应用
- 测试 Android 应用
- 多开社交媒体应用
