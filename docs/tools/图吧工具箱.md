# 图吧工具箱

## 功能简介

图吧工具箱是一款开源、免费、绿色、纯净的硬件检测工具合集，专为所有计算机硬件极客、DIY爱好者、各路大神及小白制作。

## 主要特点

- 集成众多实用硬件检测工具
- 开源免费且绿色无污染
- 界面简洁易用
- 支持各种硬件平台
- 持续更新和维护

## 使用场景

- 硬件信息检测
- 电脑故障诊断
- 硬件性能测试
- DIY装机辅助
