# Process Lasso

## 功能简介

Process Lasso 是一款独特的调试进程级别的系统优化工具，通过动态调整各个进程的优先级来实现为系统减负的目的。

## 主要特点

- 动态调整进程优先级
- 防止系统假死和蓝屏
- 减少进程占用 CPU 时间过多的问题
- ProBalance 技术优化系统响应
- 进程黑名单和白名单管理
- 实时监控系统进程状态

## 使用场景

- 提升系统整体响应速度
- 防止游戏或应用卡顿
- 优化后台进程资源占用
- 避免系统无响应问题
