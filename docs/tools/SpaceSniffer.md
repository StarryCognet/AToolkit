# SpaceSniffer

## 功能简介

SpaceSniffer 是一款免费的磁盘空间分析工具，可以帮助用户快速识别和清理占用大量空间的文件。

## 主要特点

- 图形化的磁盘空间可视化显示
- 方块大小表示文件和文件夹大小
- 实时更新磁盘空间使用情况
- 支持过滤和搜索功能
- 简洁直观的用户界面
- 便携式应用，无需安装

## 使用场景

- 查找占用大量空间的文件和文件夹
- 磁盘空间清理和优化
- 分析存储使用分布
- 快速定位不需要的大型文件
