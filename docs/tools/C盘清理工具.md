# C盘清理工具 (联想)

## 功能简介

C盘清理工具是一款对 C盘进行深度清理的工具，还提供更改虚拟内存等功能。

## 主要特点

- C盘空间深度清理
- 虚拟内存设置和优化
- 系统垃圾文件清理
- 注册表清理优化
- 启动项管理

## 使用场景

- 清理 C盘存储空间
- 优化系统性能
- 解决 C盘空间不足问题
- 系统维护和优化
