# LastActivityView

## 功能简介

LastActivityView 是一款可以查看电脑自使用以来所有浏览记录的工具，能够记录文件路径和网址等信息。

## 主要特点

- 记录系统活动历史
- 显示文件访问记录
- 记录网址浏览历史
- 按时间排序显示活动
- 支持导出活动报告
- 轻量级绿色软件

## 使用场景

- 系统使用情况审计
- 查看近期操作历史
- 恢复误关闭的文件路径
- 监控电脑使用情况
