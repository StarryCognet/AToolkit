# zTasker

## 功能简介

zTasker 是一款 Windows 上的万能计划任务工具，能通过定时、CPU 占用、文件夹监控、网速、快捷键等多种条件运行指定任务和程序。

## 主要特点

- 多种触发条件（定时、CPU、文件夹监控等）
- 丰富的任务类型（程序运行、文件操作等）
- 图形化配置界面
- 支持快捷键触发
- 网络状态监控
- 文件备份功能

## 使用场景

- 自动化系统维护任务
- 定时执行程序或脚本
- 文件夹变化监控和处理
- 网络状态自动响应
- 系统资源优化调度
