# WizTree

## 功能简介

WizTree 是一款适用于 Windows 的快速磁盘空间分析工具，可以快速扫描硬盘并显示哪些文件和文件夹占用了最多的磁盘空间。

## 主要特点

- 极速磁盘空间分析
- 直接读取主文件表(MFT)，绕过 Windows 文件系统
- 图形化文件列表显示
- 支持按大小排序文件和文件夹
- 可作为便携式应用使用
- 占用系统资源少

## 使用场景

- 查找占用大量存储空间的文件
- 磁盘空间清理和优化
- 分析存储使用模式
- 快速定位不需要的大文件
