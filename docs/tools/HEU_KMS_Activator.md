# HEU_KMS_Activator

## 功能简介

HEU_KMS_Activator 是一款适用于 Windows 和 Office 全系列版本的激活工具，无需联网即可离线一键全自动激活。

## 主要特点

- 支持 Windows 和 Office 所有版本
- 离线一键全自动激活
- 基于 KMS 服务端模拟环境
- 简洁高效的操作界面
- 激活后支持定期自动续期

## 使用场景

- Windows 系统激活
- Microsoft Office 套件激活
- 企业批量激活管理
- 教育机构软件授权
