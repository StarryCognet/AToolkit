# ManicTime

## 功能简介

ManicTime 是一款时间管理工具及电脑使用记录软件，通过它可以很容易分析出自己是如何使用时间的。

## 主要特点

- 全面记录电脑使用情况
- 详细的时间分析报告
- 应用程序使用时间统计
- 网站访问记录追踪
- 可视化时间使用图表
- 支持手动添加时间记录

## 使用场景

- 个人时间管理分析
- 工作效率评估
- 项目时间跟踪
- 习惯养成监督
