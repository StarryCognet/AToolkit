# 格式工厂 (Format Factory)

## 功能简介

格式工厂是一款免费的格式转换软件，可以将音频、视频、图片文件进行格式转换，满足用户的多样化需求。

## 主要特点

- 支持几乎所有主流音视频格式转换
- 图片格式转换功能
- 支持 DVD、CD 等光盘抓轨
- 支持 iPod/iPhone/PSP 等设备专用格式
- 支持常用视频、音频、图片合并功能
- 支持水印添加和特效功能

## 使用场景

- 音视频格式转换
- 设备适配格式转换
- 媒体文件压缩
- 格式标准化处理
