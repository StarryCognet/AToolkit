# 鼠大侠

## 功能简介

鼠大侠是一款电脑连点器软件，设置简单、功能强大，每秒可实现100次自动点击操作。

## 主要特点

- 高频自动点击（每秒100次）
- 简单易用的设置界面
- 支持鼠标左键、右键点击
- 可设定点击间隔时间
- 支持热键启动和停止

## 使用场景

- 游戏辅助（如挂机刷材料）
- 自动化重复点击任务
- 网页或软件自动化操作
- 减轻手动重复操作负担
