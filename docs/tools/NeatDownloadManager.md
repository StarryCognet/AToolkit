# Neat Download Manager

## 功能简介

Neat Download Manager（NDM）是一款免费轻量小巧的多线程下载工具，支持多线程下载和在线视频嗅探下载。

## 主要特点

- 多线程下载，提高下载速度
- 在线视频嗅探和下载功能
- 支持断点续传
- 浏览器集成插件支持
- 简洁直观的用户界面
- 支持 HTTP、HTTPS、FTP 协议

## 使用场景

- 快速下载大型文件
- 在线视频下载
- 替代付费下载工具（如 IDM）
- 日常网络资源下载管理
