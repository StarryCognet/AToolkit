# PanTools

## 功能简介

PanTools 是一款针对多个热门网盘的文件管理工具，提供丰富的批量操作功能。

## 主要特点

- 多网盘文件管理
- 批量分享和转存
- 批量复制和重命名
- 批量链接检测
- 跨账号移动文件
- 多账号文件搜索
- 支持不同网盘间操作

## 使用场景

- 网盘文件批量管理
- 资源整理和分类
- 多账号协同工作
- 网盘资源迁移
