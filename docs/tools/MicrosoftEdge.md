# Microsoft Edge

## 功能简介

Microsoft Edge 是微软开发的浏览器，也是现在 Windows 电脑自带的浏览器。

## 主要特点

- 基于 Chromium 内核
- 与 Windows 系统深度集成
- 性能优化和安全性提升
- 支持 Chrome 扩展程序
- 集成 Cortana 助手
- 节省内存和电池续航

## 使用场景

- 日常网页浏览
- 在只有流氓浏览器的电脑上使用
- 需要与微软生态系统集成的工作环境
- 对浏览器安全性有较高要求的场景
