# Dism++

## 功能简介

Dism++ 是一款 Windows 系统优化工具，轻巧、好用、不流氓，堪称强迫症晚期患者福音。

## 主要特点

- 系统精简和定制
- 补丁管理
- 启动项管理
- 服务优化
- 注册表清理
- 系统垃圾清理
- 绿色便携无需安装

## 使用场景

- C盘深度清理
- 系统性能优化
- 定制个性化系统
- 清理系统垃圾文件
- 管理系统服务和启动项
