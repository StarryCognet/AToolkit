# FastCopy

## 功能简介

FastCopy 是 Windows 上最快的文件复制和备份软件，以其卓越的性能著称。

## 主要特点

- 单线程文件复制速度可达硬盘极限
- 支持多种复制模式（完全验证、同步删除等）
- 支持过滤器和包含/排除规则
- 支持长文件路径（超过 260 个字符）
- 支持 Unicode 和多字节字符
- 可以从命令行调用

## 使用场景

- 大量数据的快速复制和备份
- 系统迁移和镜像制作
- 文件同步和归档
- 紧急数据恢复和转移
