# Everest (AIDA32)

## 功能简介

EVEREST（原名AIDA32）是一个功能强大的测试软硬件系统信息的工具，它可以详细显示 PC 每一个方面的信息。

## 主要特点

- 支持超过 3400 种主板型号检测
- 支持超过 360 种显卡型号检测
- 支持并口/串口/USB 等 PNP 设备检测
- 支持各类处理器的详细信息侦测
- 集成 Windows 系统几乎所有硬件和软件信息查询功能

## 使用场景

- 系统硬件详细信息查询
- 故障诊断和硬件兼容性检查
- 系统性能基准测试
- 硬件资产管理和审计
