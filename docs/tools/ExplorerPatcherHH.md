# ExplorerPatcherHH

## 功能简介

ExplorerPatcherHH 是一款开源工具，主要用于让 Windows 11 的开始菜单重新变回 Windows 10 样式，并且支持磁贴功能。

## 主要特点

- 将 Windows 11 开始菜单改为 Windows 10 样式
- 支持传统开始菜单磁贴功能
- 兼容 Windows 10 和 Windows 11
- 开源免费
- 高度可定制

## 使用场景

- 不喜欢 Windows 11 新界面的用户
- 需要传统开始菜单磁贴功能
- 希望统一不同 Windows 版本界面风格
- 提高工作效率和操作习惯延续
