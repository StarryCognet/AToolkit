# CrystalDiskInfo

## 功能简介

CrystalDiskInfo 是一款硬盘（包括 SSD）的实用软件，可以查看硬盘的详细信息，如型号、容量、缓冲区大小等，并监视磁盘的健康状况。

## 主要特点

- 显示硬盘基本信息：采集型号、容量、缓冲区大小等
- 查看 S.M.A.R.T. 信息：电源启动次数、使用时间、温度等
- 实时监控磁盘健康状态
- 支持多种硬盘接口（SATA、NVMe 等）
- 提供二次元人物界面（Shizuku 版本）

## 使用场景

- 检查硬盘健康状况，预防数据丢失
- 监控硬盘温度，避免过热损坏
- 查看硬盘使用时间和电源循环次数
- 评估 SSD 寿命和剩余写入寿命
