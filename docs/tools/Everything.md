# Everything

## 功能简介

Everything 是 Windows 上一款搜索引擎，能够基于文件名快速定位文件和文件夹位置。

## 主要特点

- 实时文件名搜索
- 极快的搜索速度
- 支持正则表达式搜索
- 支持网络搜索
- 轻量级且资源占用少
- 支持 HTTP 和 ETP 协议共享

## 使用场景

- 快速查找文件和文件夹
- 在大量文件中定位特定文件
- 替代 Windows 自带搜索功能
- 提高文件管理效率
