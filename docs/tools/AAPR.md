# Advanced Archive Password Recovery

## 功能简介

Advanced Archive Password Recovery 是一款用于恢复任何丢失或忘记的 ZIP 压缩包密码的工具，采用暴力破解方式。

## 主要特点

- 支持多种压缩格式密码恢复
- 暴力破解和字典攻击模式
- 支持自定义字符集
- 多线程加速破解过程
- 断点续破功能

## 使用场景

- 恢复遗忘的压缩包密码
- 获取重要的受密码保护文件
- 数据恢复和取证分析
