# Watt Toolkit (Steam++)

## 功能简介

Watt Toolkit（原名 Steam++）是一款专为游戏和平台网页设计的加速器工具。

## 主要特点

- Steam 商店和社区加速
- Epic、Origin、Uplay 等游戏平台加速
- GitHub、Google 等网站加速
- 网络代理功能
- Hosts 管理
- 游戏工具箱（解锁帧率限制等）

## 使用场景

- 加速 Steam 商店访问和游戏下载
- 解决国外游戏平台连接问题
- 加快 GitHub 等开发者网站访问速度
- 绕过某些网络限制
