# CrystalDiskMark

## 功能简介

CrystalDiskMark 是一款专业的磁盘性能测试工具，用于评估机械硬盘和固态硬盘（SSD）的读写速度和随机 I/O 性能，也就是常说的硬盘测速软件。

## 主要特点

- 测试顺序读写性能
- 测试随机读写性能（4K、512K 等）
- 支持多种测试模式和数据块大小
- 图形化界面，易于操作
- 提供二次元人物界面（Shizuku 版本）

## 使用场景

- 购买新硬盘前后的性能对比
- 评估硬盘是否达到标称性能
- 检测硬盘老化导致的性能下降
- 对比不同硬盘或不同接口的性能差异
