# Auto Macro Recorder

## 功能简介

Auto Macro Recorder 是一款功能强大的自动化应用编写工具，可以通过录制键盘/鼠标操作来记录和播放重复行为。

## 主要特点

- 键盘和鼠标操作录制
- 自动化脚本回放
- VBA 代码生成
- 支持条件判断和循环
- 可编辑和自定义宏脚本
- 简单易学的界面

## 使用场景

- 自动化重复性工作任务
- 批量数据录入
- 软件测试自动化
- 游戏辅助操作
- 提高工作效率
