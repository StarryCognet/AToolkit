# Geek Uninstaller

## 功能简介

Geek Uninstaller 是一款干净的去除和强制去除软件工具，执行深度和快速扫描，并删除所有剩余部分，保持电脑清洁。

## 主要特点

- 深度卸载功能
- 强制卸载顽固软件
- 快速扫描残留文件
- 清理注册表垃圾
- 简洁直观的界面
- 轻量级绿色软件

## 使用场景

- 彻底卸载不需要的软件
- 清理软件残留文件和注册表项
- 释放磁盘空间
- 保持系统整洁
