# oCam

## 功能简介

oCam 是一款设计简单但功能实用的免费屏幕截图/屏幕录像/音频录制软件。

## 主要特点

- 屏幕录制功能
- 屏幕截图功能
- 音频录制支持
- 全屏或自定义区域录制
- 支持系统声音和麦克风录音
- 多种输出格式支持
- 简洁直观的用户界面

## 使用场景

- 制作教学视频
- 游戏录制和分享
- 屏幕操作演示
- 在线会议记录
- 软件功能展示
