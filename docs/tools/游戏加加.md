# 游戏加加 (GamePP)

## 功能简介

游戏加加是一款帮助用户提升电脑性能、有效改善游戏体验的综合性软件，为用户提供专业的电脑硬件检测。

## 主要特点

- 游戏性能优化
- 硬件实时监控
- 游戏录像和截图
- 硬件信息检测
- 游戏辅助功能
- 帧数监控和优化建议

## 使用场景

- 提升游戏运行性能
- 监控游戏时系统状态
- 记录游戏精彩瞬间
- 优化电脑游戏体验
- 硬件性能评估
