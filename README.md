# AToolkit

一个基于 VitePress 的现代化文档工具包。

## 简介

AToolkit 是一个使用 VitePress 构建的强大而灵活的文档工具包项目。它专为开发者、技术写作者以及需要搭建轻量级文档网站的用户而设计，提供了一套完整的解决方案来创建、管理和发布技术文档。

该项目利用现代前端技术栈，基于 VitePress 构建，具有出色的开发体验和快速的构建速度。AToolkit 不仅可以用于创建软件文档，还可以作为知识库、个人笔记系统甚至是小型博客平台。

## 网站用途与分享内容

### 网站用途

AToolkit 是一个专注于分享实用工具和资源的平台，旨在帮助开发者和技术爱好者：

我们致力于分享以下类型的内容：

## 文档组织规范

AToolkit 采用结构化的文档组织方式，按照预定义的分类进行文档管理。每个分类对应一个独立的 Markdown 文档，文档名称遵循"[分类名]类.md"的命名格式。

主要特点：
- 所有具体工具文档被删除，仅保留分类文档
- 使用统一的 ToolList 组件展示工具列表
- 创建全局导航入口文件[目录.md]，采用网格布局展示所有分类
- 包含分类名称、工具数量和代表性工具预览

## 快速开始

### 安装依赖

```bash
pnpm install
```

### 开发模式

```bash
pnpm docs:dev
```

启动本地开发服务器，通常在 http://localhost:5173 上访问。

### 构建生产版本

```bash
pnpm docs:build
```

构建静态文件到 `docs/.vitepress/dist` 目录。

### 预览生产版本

```bash
pnpm docs:preview
```

在本地预览构建后的站点。

## 技术架构

AToolkit 基于以下技术理念构建：

### 核心架构
- 使用 VitePress 作为文档框架，充分利用其基于 Vite 和 Vue 3 的高性能特性
- 采用纯前端静态生成（SSG）模式，确保部署简单且性能优异
- 所有文档内容由 VitePress 自动处理路由与渲染

### 主题定制
- 自定义主题自动加载位于 `.vitepress/theme` 目录
- 全局视觉效果组件注入到 `layout-top` 插槽
- 统一应用渐变色方案，使用 CSS 变量管理样式

### 响应式设计
- 支持桌面端、平板和手机等多种设备
- 桌面端采用多列网格布局
- 平板端优化为 2 列显示
- 手机端采用单列布局确保可读性

### 性能优化
- 静态资源优化处理
- 图片懒加载策略
- CSS 和 JavaScript 压缩
- 现代浏览器特性优先策略

## 开发规范

### 侧边栏配置
- 使用 `vitepress-sidebar` 自动生成侧边栏
- 正确设置文档根目录的相对路径
- 默认展开菜单项以提高可发现性

### 组件开发
- 客户端代码访问全局对象时使用 `window` 并包裹环境检查
- 避免重复注册已在布局中使用的组件
- 实现渐进增强的交互体验

### 第三方服务集成
- 集成第三方服务时考虑网络不稳定情况下的容错能力
- 提供友好的错误提示界面
- 实现手动重新连接机制

## 部署指南

AToolkit 构建的站点是纯静态的，可以轻松部署到各种托管平台上：

- GitHub Pages
- Netlify
- Vercel
- 任何支持静态文件托管的服务

只需将 `docs/.vitepress/dist` 目录中的构建产物上传到服务器即可。
